# sudachiclone-wasm-demo

this is [sudachiclone-rs](https://github.com/Yasu-umi/sudachiclone-rs) wasm demo.

## build index.d.ts

build `.d.ts` for wasm

```bash
yarn run build-tsd
```

## run dev-server

start webpack-dev-server

```bash
yarn run dev
```

## deploy

deploy to [github pages](https://yasu-umi.github.io/sudachiclone-wasm-demo/)

```bash
yarn run deploy
```
